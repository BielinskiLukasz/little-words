<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| navigator.userAgent → install prompt logic | User agent string used to detect iOS; can be spoofed by user |
| localStorage → iOS prompt seen flag | Persisted flag; can be cleared by user (expected behavior) |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation |
|-----------|----------|-----------|----------|-------------|------------|
| T-02-05-T1 | Tampering | navigator.userAgent spoof | low | accept | User agent check is a convenience UX feature (install prompt), not a security gate; spoofing only affects whether the user sees the prompt — no data or access risk |
| T-02-05-I1 | Information Disclosure | localStorage little-words-ios-prompt-seen | low | accept | Flag is a boolean string ('true'); no personal data stored; origin-isolated by browser |
| T-02-05-I2 | Information Disclosure | import.meta.env.VITE_APP_VERSION in About section | low | accept | App version is public information; visible in built JS bundle regardless; no security impact |
| T-02-05-D1 | Denial of Service | language change on every Settings render | low | mitigate | i18n.changeLanguage() only called on click (not on render); localStorage write is synchronous but negligible; no performance issue |
</threat_model>
