| Threat ID | Category | Component | Severity | Disposition | Mitigation |
|-----------|----------|-----------|----------|-------------|------------|
| T-02-03-T1 | Tampering | OnboardingWizard form inputs | low | mitigate | zod schema validates all fields before saveChildProfile is called; React escapes output by default |
| T-02-03-T2 | Tampering | language custom text input | low | mitigate | Custom language text is stored as plain string; no SQL/query injection risk with IndexedDB; max characters not enforced but data is local-only |
| T-02-03-I1 | Information Disclosure | child name in WelcomeScreen URL | low | accept | /#/ hash routing does not expose child name in the URL; name is only in the component props |
