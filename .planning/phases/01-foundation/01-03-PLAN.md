NOTE: Polish plural rules (CLDR): count=1 → _one; count 2–4,22–24,… → _few; count 5–21,25–31,… → _many; fractional → _other. i18next resolves these automatically using Intl.PluralRules. The test MUST verify count=2 returns 'znaczenia' (the _few form) — this is Success Criterion 3.
